## 🚀 Installation (3 Easy Steps)

### Step 1: Install Dependencies

```powershell
npm install
```

This installs:
- React 18 + TypeScript
- Vite (super fast builds)
- Tailwind CSS
- Framer Motion (animations)
- React Helmet (SEO)
- React Scroll
- React Icons
- Axios

### Step 2: Swap HTML Files

```powershell
# Backup old file
Rename-Item index.html index-old.html

# Use new React file
Rename-Item index-new.html index.html
```

### Step 3: Start Development

```powershell
npm run dev
```

Open http://localhost:3000 - Your new site is live! 🎉
